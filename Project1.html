Welcome To Rock Paper Scissor Game

Instruction:
  1. Enter the Username if kept empty it choose "Player" as name.
  2. Enter Match Point if not Default match point is 5.
  3. Click on Start Button To start.
  4. If want to stop in between Click on stop button.
  5. click on reset button to Reset to original setting.
Rules to Play:
  1. Click on any option to select you choice.
  2. Computer will also choose randomly any option.
  3. Result will be shown in the given Screens.
  4. Result will be decides as:
    • Rock beats Scissor.
    • Scissor beats Paper.
    • Paper beats Rock.
  5. Start Playing and Enjoy.
